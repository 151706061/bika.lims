Installation
------------

### Install Plone:

Download the latest Unified Installer from http://plone.org/download and
visit http://plone.org/documentation/topic/Installation for more about
installing and configuring Plone.

### Install Gnuplot:

Gnplot is required for plotting results.

Visit http://gnuplot.sourceforge.net/

### Install Bika LIMS:

Edit buildout.cfg and add "bika.lims" to the eggs section:

    eggs =
        Plone
        Pillow
        bika.lims

Run buildout:

    $ bin/buildout

### Start Plone:

    $ bin/plonectl fg

### Add a new site:

Visit http://localhost:8080 and add a new site.  Be sure to select the
checkbox for the Bika LIMS extension profile.

### Test:

Visit http://localhost:8080 and view your site.
